For my collage project, used an original image of Doc McStuffins, 
who is the main character in a childhood favorite show of mine. 
For this project, I used the original Doc McStuffins image and placed 
six copies of it on to one canvas. The first copy was of the original 
image, and the other 5 copies are different manipulations of that same 
image. I changed one of the copies to grayscale by averaging the red, 
green, and blue color values of each pixel. I made another copy sepia 
by first converting it to grayscale and then adjusting the gray colors 
in order to create darker shadows, brown mid-tones, and lighter 
highlights. I also created a negated version of the image by subtracting
each pixel's red, green, and blue color values from 255, which the 
highest value for a color. For my recursive manipulation of the original 
image, I repeatedly copied smaller versions of the image inside of itself 
in order to create a layered recursive effect. For the required mirroring 
transformation, I horizontally mirrored my original Doc McStuffins image by 
copying the left side of the image on to the right side of the image. 
Finally, after placing all 6 versions of the image on to the same canvas, 
I saved my final finished collage as an image file called IGunjaFinalCollage.jpg.
